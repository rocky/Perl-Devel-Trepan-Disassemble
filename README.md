Synopsis
--------

Adds a "disassemble" command to [`Devel::Trepan`](https://github.com/rocky/Perl-Devel-Trepan/wiki).

Installation
------------

To install this Devel::Trepan, run the following commands:

	perl Build.PL
	make
	make test
	[sudo] make install
